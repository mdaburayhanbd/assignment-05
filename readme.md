1. var is function-scoped and allows redeclaration, let is block-scoped and allows reassignment, and const is block-scoped and cannot be reassigned.

2. The spread operator (`...`) expands elements of an array, object, or iterable into individual items for copying, merging, or passing arguments.

3. map() returns a new array by transforming each element, filter() returns a new array with elements that meet a condition, and forEach() just executes a function on each element without returning anything.

4. An arrow function is a shorter way to write functions in JavaScript using the => syntax, which automatically binds this to its surrounding context.

5. Template literals are strings enclosed by backticks (`) that allow embedded expressions with ${} and support multi-line text easily.
